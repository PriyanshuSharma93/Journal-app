# Journal App

A backend Journal Application built using Spring Boot and MongoDB.

## 🚀 Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven

## ✨ Features
- User Management
- Create Journal Entries
- Update Journal Entries
- Delete Journal Entries
- REST APIs

## 🛠️ Future Enhancements
- JWT Authentication
- Spring Security
- Swagger Documentation
- Docker
- Deployment
